# Expense Tracker

## Description
A web-based Expense Tracker application built using HTML, CSS, and JavaScript.  
Track your daily, monthly, and yearly expenses with options to update and delete entries.

## Features
- Add, update, and delete expenses
- View daily, monthly, and all-time expenses
- Calculate totals per day, month, and year
- Categorize expenses by type and source

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: (Optional future Java + MySQL)
- Storage: LocalStorage (frontend)

## How to Run
1. Clone or download the repository
2. Open `index.html` in a web browser
3. Start adding your expenses

## Key Expertise
- HTML, CSS, JavaScript
- DOM Manipulation
- LocalStorage for persistent data
- UI/UX design for simple web apps
